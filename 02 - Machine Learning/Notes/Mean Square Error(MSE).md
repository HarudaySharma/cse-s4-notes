 **MSE** is the average squared loss per example over the whole dataset.

>[!important] formula
> $$\frac{1}{n} \sum_{i=1}^{D}(y_i-\hat{y}_i)^2$$
> - $n =>$ number of examples
> - $y =>$ actual value of label
> - $\hat{y} =>$ value model predicts for $y$
